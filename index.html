Welcome To My Portfolio

I focus on working with real-world operational data—cleaning it, validating it, and turning it into systems that support analysis and decision-making. My background spans roles where data was often messy, incomplete, and underutilized, which naturally pushed me toward analytics and system-building work.

While much of my early experience comes from the green industry—including horticulture, irrigation, and asset management—the projects showcased here emphasize transferable skills: SQL-based data cleaning, exploratory analysis, geospatial mapping, and data visualization. The domain varies, but the underlying approach remains the same.

Throughout my career, I have been consistently drawn to the data itself—particularly how better structure and tooling can improve outcomes. My first informal project was an automated Excel workflow that streamlined operational ordering, which ultimately set me on the path toward data analysis and systems-oriented work.

This portfolio includes several projects with completed portions as well as ongoing work, including a large-scale public dataset analysis of a Washington, DC tree survey.

Skills

  • Excel - Building and analyzing spreadsheets for sales data, inventory management, and financial reporting.
  • SQL - Using MySQL and SQLite for data cleaning, validation, and exploratory analysis.
  • QGIS - Creating high-level geospatial maps for tree inventories, private infrastructure, and property management.
  • Tableau - Designing visualizations, dashboards, and reports for both summary-level and detailed analysis.
  • HTML - Working with basic HTML structure and customization. This website is built from a template that I adapted to support my long-term portfolio needs.
